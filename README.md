# Marks Whiteboard Extension

## Project Documentation

### Introduction
This project is a Chrome extension designed to enhance online learning experiences by providing a virtual whiteboard for note-taking, drawing, and collaboration.

### Features
- **Interactive Drawing:** Draw and annotate on a virtual whiteboard in real-time.
- **Collaboration:** Share your whiteboard with classmates and work together.
- **Export Options:** Save your whiteboard as an image or text.

### Installation
1. Download the extension files from the repository.
2. Go to Chrome extensions page (chrome://extensions/).
3. Enable "Developer mode".
4. Click on "Load unpacked" and select the downloaded folder.

### Usage
- Click the extension icon to open the whiteboard.
- Use the tools provided to create and collaborate.
